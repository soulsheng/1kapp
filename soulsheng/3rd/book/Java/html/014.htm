开发平台-j2sdk的安装与配置　

　　下载j2sdkhttp://java.sun.com/j2se/1.4.1/download.html
　　双击j2sdk-1_4_2-windows-i586可执行程序

　

　　桌面显示安装向导准备工作对话框　

　　

　　等准备工作完成以后，显示“许可协议”对话框，用户只需选择“I accept the terms in the license aggrement”，点击“next”按钮。　

　　系统显示“Custom Setup”对话框，在此用户可以选择需要安装的内容（建议刚开始按照缺省设置），修改安装目录。　

　　桌面显示浏览器注册对话框　

　

　　系统开始自动完成安装过程　　

　　

　　如果安装成功。系统显示完成对话框　

　

　　此时，能够在桌面和“开始”/“所有程序”中看到“Java Web Start”图标，说明安装过程已经完成　　

j2sdk环境配置　

　　在桌面上选中“我的电脑”，点击右键，然后点击“属性”进入“系统属性”对话框

　　选择“高级”/“环境变量”，进入“环境变量”对话框

　　点击“环境变量”对话框中的“系统变量”下的“新建”按钮，进入“编辑系统变量”对话框


编译和运行Java程序　

　　选择“命令提示符”，进入DOS状态并进入文件存放的位置

　　输入编译命令，如果编译成功，执行该程序。

JCreator的安装与配置　　

　　下载JCreator工具软件http://www.jcreator.com(共享软件)
　　按照安装向导自动完成按照
　　启动JCreator，显示主界面
　　选择“Configure”/“options…”，弹出“Options”对话框

　

　　选择“JDK Profiles”，点击“New”按钮，弹出“Select path”对话框，在此对话框中选择j2sdk安装的文件夹，然后点击“确定”按钮。　　

　　

　　弹出“JDK Profile”对话框，显示已经配置的内容，点击“OK”按钮，完成JCreator工具的配制。

　

　　此时，再次显示“Options”对话框，显示已经配置了j2sdk11.4.2　

　　
　　打开ch1ex2.java程序　　

　　
　　选择菜单栏中的“Build”/“Compiler”编译程序，如果程序没有错误，在窗口的下面显示“Process Completed”，如果有错误，则显示错误的语句、所在位置以及错误性质。　　


　　
　　然后，选择菜单栏中的“Build”/“Execute File”执行程序，系统在另一个窗口中显示运行结果。　　

　　
　　如果是新建一个java文件，则选择菜单栏中的“File”/“New”，弹出“New”对话框。在此对话框中选择“Files”，并分别在Filename和Location输入文件名和选择存放位置。
