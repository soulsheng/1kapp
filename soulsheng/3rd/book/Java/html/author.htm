丁岳伟教授

　　丁岳伟教授，1961年生。现任上海理工大学计算机工程学院院长、教授。主要从事计算机网络应用、信息安全，软件工程和CMM等方面的研究和教学工作。近年来，支持并参加了上海市重点攻关和科委软课题等10多项项目，发表论文20多篇，编著教材５部。曾获得上海市高校优秀青年教师、上海市育才奖、机械工业部教书育人奖和上海市决策咨询研究成果三等奖等。

 

彭敦陆博士

　　 彭敦陆,男,1974年生,1999年毕业于南京大学,获理学硕士学位。现为复旦大学计算机软件与理论专业在读博士,上海理工大学讲师。长期从事数据库及数据挖掘技术、Web信息管理技术、XML技术、GIS及CMM方面的研究和教学工作。开设数据库原理、java语言程序设计、分布式系统、XML高级理论及应用等本科生和研究生课程。曾主持或参加过多项国家自然基金、国家“863”项目、国家“973”项目及上海市教委基金项目。在各类期刊及国内外学术会议上发表论文十多篇。于2004年被评为“上海市优秀青年教师后备人选”。
