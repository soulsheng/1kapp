概述

　　Java语言初期是利用一种窗口开发类库（Abstract Windows Toolkit，简记为AWT）实现图形用户界面，AWT提供了基本的用户接口组件（component），如标签（label）、菜单（menu）、按钮（button）等，以及剪贴板、数据库转换、图形与图像操作等，另外也提供事件处理机制。
　　到了Java2以后，推出了功能更加强大、用途更广、使用更加灵活的一个Java基本类库（Java Foundation Classes ，简记为JFC），而其GUI部分则称作为Swing。
　　大部分的Swing组件是直接使用Java字节码在“画布（canvas）”上绘制，它使用很少操作环境的本地GUI资源。
　　并非AWT类库已完全被Swing类库所取代，AWT只是部分被Swing所取代而已。当我们想使用Swing时，仍然需要与AWT的包及其类一同配合来使用。
