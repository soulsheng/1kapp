课程的定位和目标　

　　总体目标
　　　　要求学生理解程序设计的基本概念，强调工业标准化编程实践的必要性。
　　　　介绍软件设计的基本原理，提供构造可靠软件的必要训练。
　　　　如何利用程序设计语言构造现实世界数据以及电子数据的处理。
　　具体目标
　　　　使用Java语言创建Java程序。
　　　　熟练掌握一种开发平台。
　　　　描述和使用Java语言面向对象的特点。
　　　　描述和使用Java语言的事件处理模式。
　　　　实现程序的输入、输出以及文件操作。
　　　　利用AWT和Swing类库开发图形用户界面。
　　　　使用Java语言的异常处理机制控制程序的执行和定义用户自己的异常处理。
　　　　使用Java语言的多线程技术，开发多线程程序。
　　　　利用Java语言的类库编制常用的查找和排序程序和定义自己处理查找和排序的算法和程序。
　　　　使用Java语言的Sockets实现网络的TCP、UDP通信。
　　　　初步掌握数据库访问技术和远程对象调用的方法以及程序结构。
　　　　掌握独立应用程序和小应用程序的开发方法。
　　　　掌握利用Java语言类库实现声音、文字、图像及动画等多媒体信息的编程。
　　　　初步了解JSP(Java Server Pages)、小服务器程序(Servlet)、JavaBeans和EJB的概念。

课程的领域范围

　　计算机类专业，包括计算机科学与技术、软件工程、网络工程。信息技术相关专业，包括电气信息工程、电子信息工程、电子信息科学与技术、通信工程、信息安全、信息工程、信息与计算科学、自动化等。

课程的主要特色

　　程序设计语言本身的语法、语义并不是最重要的内容，最重要的是帮助学生理解基本概念以及如何使用这些概念解决实际问题，即编程的艺术就是如何使用明确定义的方法构造程序。为此，本书的所有概念均通过许多细致的结构和充分的实例程序而得阐明，经验告诉我们，一个好的优选的典型实例能帮助读者澄清与Java语言特征和面向对象程序设计相关的困难概念，事实上，典型实例本身允许读者通过在实际应用中的理解和研究，更深刻地了解Java语言和面向对象问题求解的思路及其优越之处。除此之外，本书在难以程度上遵循由浅入深、循序渐进，由需求引出基本概念，将复杂的概念融入具体的、经过精心设计的实例之中。

针对的学习者群体

　　定位于高等院校培养工程型、应用型（或者称为信息技术型）计算机专业人才及其相关专业的学生以及相关领域的工程技术人员，培养读者利用面向对象的技术分析、设计和解决问题的能力，指导读者利用广泛流行的Java语言以及相关开发平台开发各种应用软件，它覆盖了编程技术和Java程序设计语言。

典型的学习时间

　　如果对象基础较差，可适当考虑增加学时，另外根据各学校每学期的周数可适当调整。
　　第一门程序设计课程
　　 64学时（16周/学期）或72学时（18周/学期），其中24或28学时上机实习。
　　第一门程序设计课程
　　 48学时（16周/学期）或54学时（18周/学期），其中12或16学时上机实习，外加一周综合性的问题求解实习。
　　已经学过C/C++
　　 32学时（16周/学期）或36学时（18周/学期），其中12或16学时上机实行。
　　已经学过C/C++
　　 16学时（16周/学期）或18学时（18周/学期），其中6或8学时上机实习，外加一周综合性的问题求解实习。
