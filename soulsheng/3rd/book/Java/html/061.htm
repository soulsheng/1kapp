面向对象程序设计概念　

　　客观世界由许多各式各样的对象组成，每种对象都有各种的内部状态和运动规律，不同对象间的相互作用和联系，构成了各种不同的系统，既构成了我们所面对的客观世界。
　　对象可大可小，例如学校是一个对象，系是一个对象，专业也是一个对象，同时，针对同一个对象，站在不同的角度，可以有不同的内部状态和行为。
　　面向对象方法将客观世界的事物抽象为对象，它具有自己的状态和行为，而对象之间通过消息的传递来协同完成一系列活动。
　　面向对象编程语言中，对象的状态由数据、变量以及相关的数据结构进行描述，行为有一系列完成特定功能的方法（过程或函数）来体现，而消息的传递就是通过参数的传递来实现。
　　例1：汽车抽象为一个对象
用变量或数据来表示汽车的状态：速度、油量、型号、生产厂商、所处位置等。
用方法表示汽车的行为：加速、刹车、换挡、倒车、转向等。
　　例2：驾驶员抽象为一个对象
状态：性别、驾龄、驾驶证、手、脚等。
行为：踩刹车、踩油门、调整方向盘等。
