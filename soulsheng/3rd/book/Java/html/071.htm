程序出错概述

　　语法错误：
　　通常指编程中违反了语言的语法和语义，例如程序中忘记了分号“;”；关键字拼写错误（关键字int、double等）；程序中使用的变量或标识符没有定义就使用；在使用变量时，没有考虑其作用域；代码块的“{”和“}”不匹配等等。
　　逻辑错误：
　　是指程序运行的结果不是原先设计、编程所希望产生的结果，这就需要用户仔细阅读分析设计过程以及程序代码，以发现设计或程序中偏离原先目标的原因并进行修改。
　　异常错误：
　　异常通常指程序运行过程中出现的非正常情况，例如用户输入错误、除数为零、需要处理的文件不存在、数组的下标超出数组范围等等。
