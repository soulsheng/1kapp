声音播放　　　

　　 声音的播放通过类Applet及其接口AudioClip实现。
　　 支持的声音文件格式有.au、.mid、.wav等。
　　 在Applet类中通过getAudioClip()方法获取声音片段，然后通过AudioClip接口的相关方法实现声音的播放。GetAudioClip()有两个重载方法：
AudioClip getAudioClip(URL url);
AudioClip getAudioClip(URL url, String name);
　　　　 前者要求给出声音文件的URL（包括基地址和文件名）；
　　　　 后者第一个参数要求给出声音文件的基地址，第二个参数要求给出声音文件的文件名。
　　　　 如果声音文件和播放该文件的Applet（.class）文件/嵌有Applet的HTML文件在同一个目录下，可以用Applet的getCodeBase()/getDocumentBase()方法来获取声音文件的基地址。
例1：与.class在同一个目录下
AudioClip au=getAudioClip(getCodeBase(), ＂bird.au＂);
例2：与html文件在同一目录下
AudioClip au=getAudioClip(getDocumentBase(),＂bird.au＂);
　　　　 不论指定的声音文件是否存在，它们都立即返回，并启动后台线程完成对声音文件的下载，这样Applet可以去执行其他任务。
　　　　通过调用AudioClip接口的下列方法来控制Applet中下载的声音片段的播放：
void loop()//循环播放声音
void play()//开始播放声音（只播放一次）
void stop()//停止播放声音片
　　 直接通过调用Applet类的play()方法来实现，通过play()方法，可以播放指定URL地址的声音文件。
void play(URL url)；
void play(RUL url,String name)；
例3：播放与Applet的.class相同基地址下的声音文件“bird.au”。
play(getCodeBase(),＂bird.au＂);
【例14-1】AudioApplet类实现对指定声音文件的播放。界面上布置了四个按钮：装载(load)、播放（play）、循环（loop）及停止（stop）。它们调用AudioClip接口的相应方法实现了上述功能。同时，我们也通过Applet类的play()方法实现了指定URL的声音文件的播放。
