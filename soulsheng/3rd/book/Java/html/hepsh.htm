高等教育出版社

　　高等教育出版社创立于1954年5月18日，是中华人民共和国教育部所属的出版全国高等教育、职业技术教育和成人教育教材的综合性的大型出版社。1983年，党和国家领导人邓小平同志亲自为高等教育出版社题写了社名。　　

　　半个世纪以来，高等教育出版社坚持为人民服务、为社会主义服务、为教育事业服务，坚持积累知识，传播文化，以书育人的办社方针，艰苦创业，努力拼搏，不断创新，竭诚服务，为我国的教材出版事业奋斗。1954年～1959年，出版了前苏联教材的中译本和部分自编教材；1960年～1966年，出版了大批自编教材和部分国外教材的中译本，满足了高等学校和中等专业学校的教学需要，实现了党中央书记处关于"教材分两步走，先解决有无，再解决提高"的要求；1978年至今，累计出版教材3万余种，包括新编教材和国外教材的中译本、影印本等，满足了普通高等教育理、工、文、农、医各科，中等职业技术学校，以及成人自学高考等各个层次各个学科的教学需要。　　

    高等教育出版社以教育、文化、科学技术为纽带，凝聚了一批又一批以著名专家学者为首，以学科带头人为骨干，以有突出贡献的中青年学者为核心的作者队伍。　　

    高等教育出版社的出版物质量优良，广获好评，众多教材颇具影响。1977年～2002年，有105种图书获国家级图书奖，405种获全国性优秀图书奖，383种获省部级图书奖。　　

    随着科技的不断发展进步，高等教育出版社先后建立了"音像出版中心"、"电子出版中心"、"多媒体出版中心"，并于1999年整合为"高等教育电子音像出版社"。至今累计出版音像制品722种，其中获得国家音像制品奖2个、提名奖4个，全国优秀教育音像制品奖39个；累计出版电子出版物756种，其中获国家电子出版物奖1个、提名奖3个，获莫必斯多媒体光盘国际大奖赛中国杯奖3个，获高等教育国家级教学成果奖12个，获全国普通高等学校优秀教材奖9个。　　

    高等教育出版社1994年被评为"全国教材管理工作先进集体"；1994年获得国家技术监督局授予"全国实施法定计量单位先进集体"的称号；1995年被新闻出版署评为良好出版社；1998年12月被中共中央宣传部、中华人民共和国新闻出版署评为"全国优秀出版社"。　　

    2002年，高等教育出版社出版图书3759 种,总印数达到9000余万册，各类出版物（包括图书、音像制品、电子出版物）共获得国家图书奖、全国高等教育优秀教材奖等奖项170种，其中获国家级、省部级一等奖45种。　　

    目前，高等教育出版社已成为具备多学科、多类型、多层次、多品种、多媒体形式出版能力，业务涉及图书、期刊、音像制品、电子出版物以及网络、电视等领域的大型综合性出版传媒集团，综合实力和竞争力不断加强，并在教材出版领域引领国内潮流，并产生了一定的国际影响。　　

    进入新世纪，高等教育出版社提出了"植根教育，弘扬文化，引领潮流，竭诚服务"新的办社理念，并制定了十年发展目标和发展战略，积极推进和实施精品战略和集团化战略，努力将高等教育出版社建成国内前列、世界知名的大型的、现代化的、综合性的教育传媒集团。

十年内与上海几个大出版社平起平坐
    高等教育出版社上海分社于2001年8月成立，以“转变观念，创新机制，形成品牌”为知指导思想。
    上海分社与高教社是分公司和总公司的关系。一方面我们要从总公司经营的总体目标，另一方面要自主经营，自负盈亏。上海分社是我们内部的名称，对外我们称为“高等教育出版社上海教材服务中心”。中国出版界专业化的集团正在形成，我们这个服务中心有两项任务，一个是发行任务，二是集团如果成立，我们可以被批准异地办社而进行出版工作。但现在仍只是发行的网点和组稿的机构。整个出版的过程，包括书号、印制等还在总社。

上海分社要成为“上亿出版社”
    我们有近期任务和远期任务，近期是生存目标，总社投资3000万，3~5年内应该可以完成总社投资的回报，经济上我们可以自立了。远期任务是总社对我们地域性出版的要求，开发上海教育出版资源。今年上海分社可以盈利200万元，这也借助于中国经济的大发展、教育大发展的外部条件。我们力争一年增长1000万销售额。我们希望在十年内能与上海几个大的出版社平起平坐，常销书达到1000个品种，有一亿以上的产值，挤入“上亿出版社”的行列。
我们的优势：
    从外部看，上海是中国经济最发达的地区之一，也是名牌高校很多的城市，我们要充分利用这个地域优势。而且我们有高等教育出版社的品牌，可以得到学校的认可，我们出书品种范围也很广，是教材的超级市场。

我们的竞争力：
教育类出版社的核心竞争力是“教材的内容、体系与形式的创新与开发能力”。按总社社长的要求，我们要抓好“开发、出版、服务”这三个中心环节。
